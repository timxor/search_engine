## Run locally

```
# java version
java -version == 17.0.16-amzn


# build
mvn clean package -DskipTests


# run
java -jar target/searchengineapp-0.0.1-SNAPSHOT.jar
```

# open ui

Open browser to:

```
http://localhost:8080/
```
